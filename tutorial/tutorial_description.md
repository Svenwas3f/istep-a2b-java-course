# Java Tutorial Description 

## Lesson 1
**Title**: Installing Java
**Topics to cover**: 
- Download the installation file using this gudie: 
    - Visit: https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html 
    - Search for the correct installer
        - Windows: Download «Windows x64 installer»
        - Linux: Download «Linux Arm 64 RPM Package», «Linux x64 Debian Package» or «Linux x64 RPM Package»
        - MacOS: Download «macOS Arm 64 DMG Installer» or «macOS x64 DMG Installer»
    - Execute the downloaded file
        - Windows: Double Click installation file (.exe)
        - Linux: Execute the dpkg –i <deb-file.deb> or sudo dnf install <rpm-file.rpm> command
        - MacOS: Double Click the installation file (.dmg)
- Download Visual Studio Code
    - Visit: https://code.visualstudio.com/
    - Search for «Download for Windows», «Download for Linux» or «Download for MacOS» and download the installation file
    - Execute the installation file
        - Windows: Double Click the installation file
        - Linux: Execute the dpkg –i <deb-file.deb> or sudo dnf install <rpm-file.rpm> command
        - MacOS: Double Click the installation file 
- Setting up Visual Studio Code
    - Visit: https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack
    - Click the install button and approve
- Initialze your package and create your first java program 
    - Create a folder where you will store all your java files of this course
    - Open this folder in Visual Studio Code
    - Trust the author of this folder
    - Create the java package
    - Create your hello world document
    - Create the following package structure al.istep.javacourse 
    - Create a folder called „lesson1“
    - Create a file called „HelloWorld.java“ in the folder „lesson1“
    - Create a basic class to print „Hello World“ in the terminal
