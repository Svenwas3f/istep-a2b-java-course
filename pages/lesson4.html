Part 1: Understanding Packages

What Are Packages?

A package is a way to organize Java classes into groups. Think of packages like folders on your computer:

  • Folder: My Documents ‚Üí Photos ‚Üí 2024
  • Package: al ‚Üí istep ‚Üí javacourse ‚Üí lesson1

Packages serve several important purposes:

  1. Organization: Keep related classes together
  2. Avoiding name conflicts: You can have a class called "Student" in package lesson3 and another "Student" in package lesson4
  3. Access control: Control who can use your classes
  4. Professional structure: Real companies organize their code with packages

Package Naming Convention

Java packages typically use reverse domain names. For example:

  • com.apple.app - Apple company
  • org.google.android - Google company
  • al.istep.javacourse - Our course

Rules for package names:

  • All lowercase letters
  • Separated by dots
  • Usually start with a reversed domain name
  • Followed by the application name and component

How Packages Match Folders

The package name MUST match the folder structure:

Package Declaration Folder Structure
package al.istep.javacourse.lesson1; al/istep/javacourse/lesson1/
package al.istep.javacourse.lesson3; al/istep/javacourse/lesson3/
package com.mycompany.games; com/mycompany/games/

Part 2: Creating Custom Packages

Step 1: Create a New Package

Let's create a new package for game utilities. Follow these steps:

  1. In your project, create this folder structure: 
    al/istep/javacourse/games/
  2. Make sure the folder path exactly matches the package name

Step 2: Create Classes in Your Package

Create a file called GameCharacter.java in the games folder:

package al.istep.javacourse.games;

public class GameCharacter {
    public String name;
    public int level;
    public int health;
    public int maxHealth;
    
    public GameCharacter(String name, int startingHealth) {
        this.name = name;
        this.level = 1;
        this.health = startingHealth;
        this.maxHealth = startingHealth;
    }
    
    public void takeDamage(int damage) {
        health -= damage;
        if (health < 0) {
            health = 0;
        }
        System.out.println(name + " takes " + damage + " damage! Health: " + health);
    }
    
    public void heal(int amount) {
        health += amount;
        if (health > maxHealth) {
            health = maxHealth;
        }
        System.out.println(name + " heals " + amount + " HP! Health: " + health);
    }
    
    public void levelUp() {
        level++;
        maxHealth += 10;
        health = maxHealth;
        System.out.println(name + " reached level " + level + "!");
    }
    
    public void displayStats() {
        System.out.println("=== " + name + " ===");
        System.out.println("Level: " + level);
        System.out.println("Health: " + health + "/" + maxHealth);
    }
}

Step 3: Use Your Custom Package

Create a new file called GameDemo.java in the lesson4 folder:

public class GameDemo {
    public static void main(String[] args) {
        // Create game characters
        GameCharacter hero = new GameCharacter("Warrior", 100);
        GameCharacter mage = new GameCharacter("Mage", 60);
        
        System.out.println("=== GAME START ===\n");
        
        hero.displayStats();
        System.out.println();
        mage.displayStats();
        
        System.out.println("\n=== BATTLE ===\n");
        
        hero.takeDamage(15);
        mage.takeDamage(20);
        
        System.out.println();
        
        hero.heal(10);
        mage.heal(5);
        
        System.out.println("\n=== LEVEL UP ===\n");
        
        hero.levelUp();
        System.out.println();
        hero.displayStats();
    }
}

Understanding the import statement:

import al.istep.javacourse.games.GameCharacter;

This line tells Java: "I want to use the GameCharacter class from the al.istep.javacourse.games package." Without this import, Java won't know where to find GameCharacter.

Important Java Packages

Part 4: Introduction to the Random Class

Why Do We Need Random Numbers?

  • Games: Dice rolls, monster attacks, treasure drops
  • Simulations: Random events in modeling
  • Testing: Testing programs with random data
  • Shuffling: Shuffling cards or playlists

Creating a Random Number Generator

The Random class is in the java.util package, so we need to import it:

package al.istep.javacourse.lesson4;

import java.util.Random;

public class RandomExample1 {
    public static void main(String[] args) {
        // Create a Random object
        Random random = new Random();
        
        // Generate random integers
        int randomNumber1 = random.nextInt(10);  // 0 to 9
        int randomNumber2 = random.nextInt(10);  // 0 to 9
        
        System.out.println("Random number 1: " + randomNumber1);
        System.out.println("Random number 2: " + randomNumber2);
    }
}

Output (varies each time):

Random number 1: 7
Random number 2: 3

Common Random Class Methods

1. nextInt(int bound) - Random integer

package al.istep.javacourse.lesson4;

import java.util.Random;

public class RandomExample2 {
    public static void main(String[] args) {
        Random random = new Random();
        
        // Roll a dice (1 to 6)
        int dice = random.nextInt(6) + 1;
        System.out.println("Dice roll: " + dice);
        
        // Random number from 1 to 100
        int number = random.nextInt(100) + 1;
        System.out.println("Random number (1-100): " + number);
        
        // Random number from 0 to 999
        int code = random.nextInt(1000);
        System.out.println("Random code (0-999): " + code);
    }
}

Important: nextInt(n) returns a number from 0 to n-1, so to get 1-6, we do nextInt(6) + 1

2. nextDouble() - Random decimal number

package al.istep.javacourse.lesson4;

import java.util.Random;

public class RandomExample3 {
    public static void main(String[] args) {
        Random random = new Random();
        
        // Random decimal between 0.0 and 1.0
        double randomDouble = random.nextDouble();
        System.out.println("Random decimal: " + randomDouble);
        
        // Random price between 0 and 100
        double price = random.nextDouble() * 100;
        System.out.println("Random price: $" + price);
    }
}

3. nextBoolean() - Random true or false

Example: Simple Number Guessing Game

package al.istep.javacourse.lesson4;

import java.util.Random;
import java.util.Scanner;

public class GuessingGame {
    public static void main(String[] args) {
        Random random = new Random();
        Scanner scanner = new Scanner(System.in);
        
        // Generate a random number between 1 and 100
        int secretNumber = random.nextInt(100) + 1;
        int guess = 0;
        int tries = 0;
        
        System.out.println("=== NUMBER GUESSING GAME ===");
        System.out.println("I'm thinking of a number between 1 and 100");
        System.out.println("Can you guess it?\n");
        
        while (guess != secretNumber) {
            System.out.print("Enter your guess: ");
            guess = scanner.nextInt();
            tries++;
            
            if (guess < secretNumber) {
                System.out.println("Too low! Try higher.");
            } else if (guess > secretNumber) {
                System.out.println("Too high! Try lower.");
            } else {
                System.out.println("Correct! You guessed it in " + tries + " tries!");
            }
            System.out.println();
        }
    }
}

Example: Dice Roller

package al.istep.javacourse.lesson4;

import java.util.Random;

public class DiceRoller {
    public static void main(String[] args) {
        Random random = new Random();
        
        System.out.println("=== DICE ROLLER ===\n");
        
        // Roll 6-sided dice 5 times
        System.out.println("Rolling a 6-sided dice 5 times:");
        int total = 0;
        for (int i = 1; i <= 5; i++) {
            int roll = random.nextInt(6) + 1;
            System.out.println("Roll " + i + ": " + roll);
            total += roll;
        }
        System.out.println("Total: " + total + "\n");
        
        // Roll 20-sided dice (D&D)
        System.out.println("Rolling a 20-sided dice:");
        int d20 = random.nextInt(20) + 1;
        System.out.println("D20 Roll: " + d20);
        
        if (d20 == 20) {
            System.out.println("Critical Hit!");
        } else if (d20 == 1) {
            System.out.println("Critical Fail!");
        }
    }
}

Part 5: Introduction to Java Documentation

What is Java Documentation?

Java documentation is a complete reference for all built-in classes and their methods. It's like a massive instruction manual for Java!

Official Java Documentation: https://docs.oracle.com/en/java/javase/17/docs/api/index.html

How to Read Java Documentation

Let's use the Random class as an example. Here's what you'll find in the documentation:

Class java.util.Random
    
Methods:
- nextInt() - Returns a random int
- nextInt(int bound) - Returns random int from 0 to bound-1
- nextDouble() - Returns random double from 0.0 to 1.0
- nextBoolean() - Returns random boolean
- nextLong() - Returns random long

Reading Method Documentation

For each method, documentation shows:

  • Method name: nextInt(int bound)
  • What it does: Returns a random integer from 0 to 'bound' minus 1
  • Parameters: bound (int) - the upper limit
  • Returns: A random int
  • Example code: How to use it

Common Java Documentation References

String Class Methods:

  • length() - Returns the number of characters
  • charAt(int index) - Returns character at position
  • substring(int start, int end) - Returns part of string
  • indexOf(String str) - Finds position of text
  • equals(String str) - Compares two strings
  • replace(String old, String new) - Replaces text

Math Class Methods:

  • Math.abs(double a) - Absolute value
  • Math.sqrt(double a) - Square root
  • Math.pow(double a, double b) - Power (a to the b)
  • Math.max(double a, double b) - Maximum of two numbers
  • Math.min(double a, double b) - Minimum of two numbers
  • Math.round(double a) - Rounds to nearest integer

Part 6: Complete Example - Interactive Game

Let's create a complete program that brings everything together!

package al.istep.javacourse.lesson4;

import java.util.Random;
import java.util.Scanner;

public class SimpleRPG {
    public static void main(String[] args) {
        Random random = new Random();
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("=== WELCOME TO SIMPLE RPG ===\n");
        System.out.print("Enter your character name: ");
        String playerName = scanner.nextLine();
        
        // Create player stats
        int playerHP = 100;
        int playerAttack = 15;
        int enemyHP = 50;
        int enemyAttack = 10;
        
        System.out.println("\nA wild monster appears!");
        System.out.println("Monster HP: " + enemyHP + "\n");
        
        // Battle loop
        int turn = 1;
        while (playerHP > 0 && enemyHP > 0) {
            System.out.println("=== Turn " + turn + " ===");
            System.out.println(playerName + " HP: " + playerHP);
            System.out.println("Monster HP: " + enemyHP + "\n");
            
            // Player attack
            System.out.print("1. Attack  2. Defend  3. Use Potion: ");
            int choice = scanner.nextInt();
            
            if (choice == 1) {
                // Random damage (8-15)
                int damage = random.nextInt(8) + 8;
                enemyHP -= damage;
                System.out.println(playerName + " attacked! Dealt " + damage + " damage!\n");
            } else if (choice == 2) {
                System.out.println(playerName + " defended! Takes half damage next turn.\n");
            } else if (choice == 3) {
                playerHP += 30;
                if (playerHP > 100) playerHP = 100;
                System.out.println(playerName + " used a potion! Healed to " + playerHP + " HP\n");
            }
            
            if (enemyHP <= 0) break;
            
            // Monster attack
            int monsterDamage = random.nextInt(6) + 5;
            playerHP -= monsterDamage;
            System.out.println("Monster attacks! Dealt " + monsterDamage + " damage!\n");
            
            turn++;
        }
        
        // Battle result
        System.out.println("\n=== BATTLE OVER ===");
        if (playerHP > 0) {
            System.out.println("Victory! " + playerName + " wins!");
        } else {
            System.out.println("Defeat! " + playerName + " was defeated!");
        }
    }
}

Common Mistakes and How to Fix Them

Mistake 1: Forgetting to import a class

Wrong:

Random random = new Random();  // ‚ùå Error: Random not found

Correct:

import java.util.Random;

Random random = new Random();  // ‚úÖ

Mistake 2: Misunderstanding nextInt bounds

Wrong thinking: nextInt(10) gives 1-10

int number = random.nextInt(10);  // Actually 0-9!

Correct: nextInt(10) gives 0-9, so for 1-10 use:

int number = random.nextInt(10) + 1;  // 1-10

Mistake 3: Package path mismatch

Wrong: File in folder al/istep/javacourse/lesson4/ but declares:

package al.istep.javacourse.lesson3;  // ‚ùå Wrong package

Correct: Match the folder structure exactly

package al.istep.javacourse.lesson4;  // ‚úÖ Matches folder

Mistake 4: Forgetting that java.lang is auto-imported

You DON'T need to import String, System, or Math:

// No import needed!
String text = "Hello";
System.out.println(text);
double root = Math.sqrt(16);
